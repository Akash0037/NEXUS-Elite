## 🎮 Overview

**Nexus Elite** is a high-performance, visually immersive gaming e-commerce platform. Every pixel is designed to deliver a premium cyberpunk aesthetic — from cinematic page transitions and 3D hero elements to interactive product cards and a real-time community hub.

Built with cutting-edge web technologies, the platform combines a bold, dark-themed UI with fluid animations, glassmorphism effects, and responsive layouts that look stunning on every device.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🖥️ **3D Hero Section** | Interactive Three.js scene on the landing page with animated geometry |
| 🎬 **Cinematic Transitions** | Full-screen energy wipe animations between every page |
| 🛒 **Smart Cart System** | Side-drawer cart with quantity management, totals, and smooth animations |
| 🏪 **The Arsenal** | Filterable product grid with Quick View modals, power meters, and ratings |
| 👕 **The Armory** | Merchandise storefront with cyberpunk card design and hover reveals |
| 🏆 **The Grid (Community)** | Interactive feed, leaderboard, trending topics, squad finder, and player spotlights |
| 📅 **Events** | Upcoming tournament and event listings |
| 🔐 **Authentication** | Login/Signup UI with cyberpunk-themed form design |
| 📬 **Contact** | Full contact form with branded layout |
| 🖱️ **Custom Cursor** | Hardware-accelerated custom cursor with hover effects |
| 🌌 **Particle Engine** | Footer particle system with glitch line effects |
| 📱 **Fully Responsive** | Mobile-first design with adaptive layouts across all breakpoints |

## 🚀 Quick Start

### Prerequisites

- **Node.js** v18+ and **npm**

### Installation

```bash
# Clone the repository
git clone https://github.com/Akash0037/NEXUS-Elite.git

# Navigate to the project
cd NEXUS-Elite

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be running at **`http://localhost:5173`**

### Production Build

```bash
npm run build
npm run preview
```

## 🎨 Design Philosophy

- **Cyberpunk Aesthetic** — Dark backgrounds, neon accents (yellow/blue), and futuristic typography using Orbitron
- **Glassmorphism** — Frosted glass panels with subtle borders and backdrop blur
- **Motion-First** — Every interaction has purpose-driven animation via Framer Motion
- **Performance** — Optimized particle systems, lazy transitions, and efficient re-renders
- **Responsive** — Fluid layouts from 320px mobile to 4K displays
